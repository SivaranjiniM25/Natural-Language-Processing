# Movie Review Sentiment Analysis with Streamlit

This project is a machine learning web application that predicts whether a movie review is **Positive** or **Negative**. It uses **Logistic Regression** and **TF-IDF Vectorization** to achieve high-accuracy sentiment classification.

* **Text Preprocessing:** Cleans HTML tags, removes stop words, and applies Lemmatization.
* **Confidence Score:** Shows how certain the model is about its prediction.
* **Interactive UI:** Built with Streamlit for a seamless user experience.
